## Addtion : - 

    a + bi
+   c + di
-------------
    p + qi   (p  = (a + c) ,q = (c + d)i)


## Subtraction : - 

    a + bi
-   c + di
-------------
    p + qi   (p  = (a - c) ,q = (c - d)i)


## Multiplication : - 

(a + bi) * (c + di)

=  ac + cb i + ad i + bd i^2 (where i^2  == -1)

=  (ac - bd) + (cb + ad)i

=  p + q i
